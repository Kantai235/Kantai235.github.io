簡介

NVActivityIndicatorView 是一系列令人驚艷的 Loading 動畫，它採用了向量繪圖的方式來呈現動畫，這是 DGActivityIndicatorView 的一個衍生，靈感來自於 Loaders.css。

  • NVActivityIndicatorView GitHub 官方網站

然後將 NVActivityIndicatorView 加入你的 Podfile 當中。
